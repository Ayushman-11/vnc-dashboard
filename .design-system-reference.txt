🎨 COLOR CODING FOR ACTIONS (CRITICAL - ALWAYS FOLLOW)
─────────────────────────────────────────────────────────────────────────────
🟣 Purple → View/Read actions (FiEye)
🟠 Orange → Warning/Investigation (FiAlertTriangle, FiShield)
🔴 Red    → Destructive actions (FiTrash2, FiX, Terminate)
🟢 Green  → Positive actions (FiCheckCircle, Resolve, Enable)
⚫ Gray   → Neutral actions (FiXCircle, Dismiss)
🔵 Blue   → Info actions (FiInfo)

🔘 ICON BUTTON PATTERN (MANDATORY FOR ALL TABLE ACTIONS)
─────────────────────────────────────────────────────────────────────────────
<button 
  className="w-10 h-10 flex items-center justify-center rounded-lg 
             bg-purple-500/20 text-purple-400 hover:bg-purple-500/30 
             transition-all group relative"
  onClick={handleAction}
  title="Action Name"
>
  <FiIcon className="text-lg" />
  <span className="absolute bottom-full mb-2 hidden group-hover:block 
                   bg-[#1a1b35] text-white text-xs py-1 px-2 rounded 
                   whitespace-nowrap z-10">
    Action Name
  </span>
</button>

Specifications:
• Size: Fixed 40×40px (w-10 h-10)
• Icon Size: text-lg (18px)
• Background: Semi-transparent color with /20 opacity
• Hover: Increase to /30 opacity
• Always include tooltip with action description

🎭 MODAL USAGE (ALL DESTRUCTIVE ACTIONS REQUIRE CONFIRMATION)
─────────────────────────────────────────────────────────────────────────────
<ConfirmModal
  isOpen={showModal}
  onClose={() => setShowModal(false)}
  onConfirm={handleConfirm}
  title="Confirm Action"
  message="Are you sure?"
  type="danger"  // warning | danger | info | success
  confirmText="Confirm"
  cancelText="Cancel"
/>

<SuccessModal
  isOpen={showSuccess}
  onClose={() => setShowSuccess(false)}
  title="Success"
  message="Operation completed successfully"
/>

🎯 CRITICAL RULES (NEVER BREAK THESE)
─────────────────────────────────────────────────────────────────────────────
✅ ALWAYS use icon-only buttons in table action columns
✅ ALWAYS include hover tooltips for icon buttons
✅ ALWAYS use confirmation modals for destructive actions
✅ ALWAYS show success modals after operations complete
✅ ALWAYS use different gradients for multiple metric cards
✅ ALWAYS use placeholder divs for hidden action buttons
✅ ALWAYS right-align action columns (justify-end)
✅ ALWAYS maintain dark theme (#0D0E1C, #17182F)
✅ ALWAYS follow color coding system
✅ ALWAYS use Feather Icons (react-icons/fi)

❌ NEVER use text buttons in tables
❌ NEVER use light backgrounds
❌ NEVER skip confirmation for destructive actions
❌ NEVER use same gradient for all cards on a page
